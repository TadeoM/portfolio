Everything is Diamonds

Fight fire as a squad of heroes.

Role: Developer and Game Designer

In this Game Jam, my team knew that we wanted to make a strategy game, and we decided on a turn-based strategy game. The name has little to do with the game, aside from the fact that we decided to use an isometric view, which makes the grid seem to be made of diamond shapes rather than squares.

In this project I worked on the grid system and making a simple A* movement behavior. The grid system was probably the most difficult part, as I wanted to make the A* able to easily reference the grid. Essentially, the A* signals the grid where it wants to move and the grid tells it where it can go, rather than the A* script doing all the work itself. The goal was for it to be a bit easier to read.
